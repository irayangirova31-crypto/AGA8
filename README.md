# AGA8 PVT Calculator

Расчет термодинамических свойств природного газа по ГОСТ Р 8.662-2009 (метод AGA8-DC92).

## Описание

Библиотека на C++ для расчета свойств природного газа:
- Фактор сжимаемости Z
- Плотность (массовая и молярная)
- Энтальпия H
- Энтропия S
- Теплоемкости Cp, Cv
- Скорость звука w
- Коэффициент Джоуля-Томсона μ
- Показатель адиабаты κ


## Сборка

```bash
mkdir build && cd build
cmake ..
make
